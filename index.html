◊ init-book-mcp

Model Context Protocol server for INIT · the framework book. Nine chapters, chapter progress tracker, SVG diagram generators — all exposed as MCP tools.

Install

npm install -g @ai-native-solutions/init-book-mcp

Wire it in

{
  "mcpServers": {
    "init-book": { "command": "init-book-mcp" }
  }
}

Tools

toolargsreturns
initbook_get_chapter{ index }{ title, content, next, prev, callout }
initbook_progress{ chapter, position? }{ saved, read, percent }
initbook_toc—{ chapters, read, percent }
initbook_diagram{ name, bloom?, label? }{ name, svg }
